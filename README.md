# Akka Sample Project with Scala for Akka Workshop

## Configure your IDE

### IntelliJ 13.0 or later
- File > Import Project ...

### IntelliJ 12
- `sbt gen-idea`
- In IntelliJ, File > Open Project

### Eclipse
- `sbt eclipse`
- In Eclipse use the Import Wizard to import Existing Projects into Workspace

### Emacs, Vi, Sublime...
- Done
